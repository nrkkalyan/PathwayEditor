Technical Requirements

  • Sun Microsystems J2SDK 1.4 or higher for software development with yFiles Complete and yFiles Viewer. Sun Microsystems J2SDK 1.3 or higher for software development with yFiles Layout and yFiles Basic.
  • Sun Microsystems JRE 1.4 or higher to execute programs that are based on yFiles Complete or yFiles Viewer. Sun Microsystems JRE 1.3 or higher to execute programs that are based on yFiles Layout or yFiles Basic.
  • A browser to view the HTML documentation.
  • An operating system that supports Java 2.

Please read the documentation for installation instructions. For programming examples have a look at the demo applications in the src/demo/ directory.
