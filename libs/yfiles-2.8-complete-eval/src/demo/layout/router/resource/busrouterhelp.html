Help

A bus is composed of one or several interconnected hub nodes and the edges that connect the regular nodes to the hubs. For clarity, the hubs and edges of the same bus are displayed in a common color.

Usage

Adding new nodes
The palette in the side panel provides two node templates, the upper one for regular nodes and the lower one for hubs. To create a node either drag its template from the side panel into the graph view or click at the desired location in the view. In the latter case, the new node's type is defined by the selected template.
Adding new edges
An edge is created by dragging the mouse from the start to the target node. Subsequently, colors are updated to reflect the changed bus structure.
Removing nodes and edges
Selected nodes and edges can be removed using the DELETE key or the corresponding icon in the tool bar.
Routing buses
The routing algorithm and its properties are available in the tool bar. Use Route All to calculate a new routing for all buses and Route Selected to route each bus in which at least one edge, one hub or one adjacent regular node is selected.

Settings

Automatic routing
Triggers an automatic routing for the connections of moved nodes, in which the remainders of the affected buses are kept fixed. New single-edge buses are routed automatically, too. Note that the routing algorithm will remove cycles and dead-end paths.
Grid
Affects both the editor and the routing algorithm. If enabled, a grid of the given spacing is shown in the view and the algorithm assigns positions on grid coordinates if possible.
Orthogonal mode
Activates the orthogonal edge creation mode which restricts edge paths of newly created edges to horizontal and vertical segments. Orthogonal edges are distinguishable by thicker lines.
Snapping
Specifies whether snap lines are activated.
